# TimeTrack

Небольшое Windows-приложение для учёта рабочего времени. Работает в системном трее и считает, сколько времени компьютер был включён и сколько из него — реально активным (были действия мыши или клавиатуры).

![Окно статистики TimeTrack](docs/stats-window.png)

## Возможности

- Учёт **общего** и **активного** времени за день
- Пауза трекинга из меню трея
- Ручной ввод часов за день (если трекер не работал)
- В таблице: время начала и окончания, отдельный столбец ручного времени (можно скрыть)
- Учёт времени по программам: Excel, Outlook, Obsidian, Chrome (колонки можно скрыть)
- Вкладка «Детализация»: сессии с датой, началом, окончанием и программой
- Сохранение выбранных фильтров между запусками
- Окно статистики: день, неделя, месяц или произвольный период (даты «С» / «По»)
- Подсказка на иконке трея с текущими итогами за сегодня
- Хранение данных в локальной SQLite-базе (`%LOCALAPPDATA%\TimeTrack\timetrack.db`)
- Portable-сборка: папка с `TimeTrack.exe` без установки и без прав администратора

Активным считается время, когда с момента последнего ввода прошло меньше 60 секунд. Паузы сна/гибернации не учитываются.

## Требования

- Windows
- Python 3.10+ (для запуска из исходников)
- Системный трей (иконка в области уведомлений)

## Установка и запуск

```powershell
cd path\to\TimeTrack
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python main.py
```

После запуска приложение появляется в трее. Двойной клик по иконке или пункт «Статистика» открывает окно с отчётом.

Либо двойной клик по `TimeTrack.bat`.

## Portable (без установки)

Подходит для ноутбука, где **нельзя устанавливать программы** и нет прав администратора.

### Зачем

Обычный запуск требует Python. Portable-версия — это готовая папка с `TimeTrack.exe`: её достаточно скопировать и запустить.

### Как собрать (на компьютере, где есть Python)

1. Откройте командную строку в папке проекта.
2. Выполните:

```bat
build_portable.bat
```

Скрипт установит зависимости сборки (включая PyInstaller) и создаст папку:

```text
portable\TimeTrack\
```

Внутри будут `TimeTrack.exe`, нужные библиотеки, файл `portable.flag` и папка `data\`.

### Как перенести на другой ноутбук

1. Скопируйте **всю** папку `portable\TimeTrack` (на флешку, в «Документы» и т.п.).
2. На другом компьютере запустите `TimeTrack.exe`.
3. Python, установщик и права администратора **не нужны**.

### Где хранятся данные в portable

Рядом с программой:

- `data\timetrack.db` — статистика
- `data\settings.ini` — сохранённые фильтры и настройки окна

Если скопировать папку целиком, данные переедут вместе с приложением.

### Важно

- Копировать нужно всю папку, а не только `TimeTrack.exe`.
- Антивирус иногда блокирует `.exe`, собранные через PyInstaller. В таком случае добавьте папку в исключения или попросите IT.

## Структура

| Файл / папка | Назначение |
| --- | --- |
| `main.py` | Точка входа, связывание трея, трекера и UI |
| `apps.py` | Определение активного окна и известных программ |
| `paths.py` | Пути к ресурсам и данным (обычный / portable режим) |
| `tracker.py` | Фоновый трекинг idle-времени и активной программы |
| `storage.py` | SQLite-хранилище дневной статистики и сессий |
| `formatting.py` | Форматирование длительности и процентов |
| `ui/tray.py` | Иконка и меню системного трея |
| `ui/stats_window.py` | Окно статистики (сводка и детализация) |
| `build_portable.bat` | Сборка portable-папки через PyInstaller |
| `TimeTrack.spec` | Конфиг PyInstaller |
| `resources/icon.png` | Иконка приложения |

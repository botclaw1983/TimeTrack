# TimeTrack

Небольшое Windows-приложение для учёта рабочего времени. Работает в системном трее и считает, сколько времени компьютер был включён и сколько из него — реально активным (были действия мыши или клавиатуры).

## Возможности

- Учёт **общего** и **активного** времени за день
- Пауза трекинга из меню трея
- Окно статистики: день, неделя, месяц или произвольный период
- Подсказка на иконке трея с текущими итогами за сегодня
- Хранение данных в локальной SQLite-базе (`%LOCALAPPDATA%\TimeTrack\timetrack.db`)

Активным считается время, когда с момента последнего ввода прошло меньше 60 секунд. Паузы сна/гибернации не учитываются.

## Требования

- Windows
- Python 3.10+
- Системный трей (иконка в области уведомлений)

## Установка и запуск

```powershell
cd path\to\TimeTrack
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python main.py
```

После запуска приложение появляется в трее. Двойной клик по иконке или пункт «Статистика» открывает окно с отчётом.

## Структура

| Файл / папка | Назначение |
| --- | --- |
| `main.py` | Точка входа, связывание трея, трекера и UI |
| `tracker.py` | Фоновый трекинг idle-времени через Win32 API |
| `storage.py` | SQLite-хранилище дневной статистики |
| `formatting.py` | Форматирование длительности и процентов |
| `ui/tray.py` | Иконка и меню системного трея |
| `ui/stats_window.py` | Окно статистики |
| `resources/icon.png` | Иконка приложения |
